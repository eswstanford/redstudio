The location module aids users in visualizing the project site on an interactive map. The module guides users through the process of identifying the project location, indicating the project site boundaries, uploading GPS data from site surveys, and uploading responses from community surveys. The uploaded data is then presented on a map, allowing users to view centralized site-related project information in an intuitive visual.


The module additionally allows users to edit site attributes on the map for a user-friendly and easily navigable interface for editing. By editing on the map, human error in editing the data in spreadsheet form may be reduced. The module includes toggling for users to control visibility of different features on the map.


The system module walks users through the process of selecting the specific technology that is appropriate for their electricity requirements and meets their site constraints. In the alpha version of Red Studio, only the microhydro option is available. Other resource types that will be available in future iterations are solar, wind, and biogas.
